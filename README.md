# 社区居民联系电话管理系统
此系统使用Spring + SpringMVC + Mybatis架构编写。数据库采用MySQL，提供强大的、完整的功能。

## 安全性
此系统在系统层面提供了众多的安全特性，产品安全无忧。这些特性包括：

* XSS安全防护
* 表单自动验证
* 输入数据过滤
* 表单令牌验证
